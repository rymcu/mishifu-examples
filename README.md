## E32CAR
- [BLECar](https://github.com/rymcu/mishifu-examples/tree/master/blecar) —— 蓝牙遥控 4WD 小车
- 控制板: `ESP32S3-DevKitC-1` 开发板
- IDE: `CLion`
- 技术栈: `PlatformIO` + `Arduino` + `h2zero/NimBLE-Arduino` + `bblanchon/ArduinoJson`
